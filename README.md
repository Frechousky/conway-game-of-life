# conway-game-of-life

Conway game of life implementation in C.

## Requires

* `make`
* `gcc`

## Optionnal

* `valgrind`

## Usage

* `make` or `make gameoflife` to build `gameoflife` binary
* `make run` to run game of life
* `make clean` to delete `gameoflife` binary
* `make valgrind` to check for memory leaks
